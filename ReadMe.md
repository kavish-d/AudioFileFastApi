## An audio file for a test

First time use:
` pip install -r requirements.txt `

Run as
` python .\src\main.py `
 or 
`uvicorn src.main:app`


`\` will redirect to `\docs` fastapi swagger docs
access new docs at `\redoc` 


Run tests by running `pytest` in root

 Written in fastapi
